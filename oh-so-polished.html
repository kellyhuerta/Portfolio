Oh So Polished

Responsive Website Design for a Local Beauty Salon

Project Overview

Challenge

The website of Oh So Polished salon did not accurately reflect the salon's brand and atmosphere, and there was no option for booking appointments through the website.

Therefore, they are looking for creating a responsive website to provide customers with better experiences on different ends and stay competitive in the neighborhood.

Objectives

  • Redesign the responsive website of Oh So Polished
  • Develop or extend coherent branding that aligns with Oh So Polished's current and/or desired clientele and customer experiences

Project Scope

Responsive Website, Branding

Tools

Figma, Adobe Illustrator, Adobe XD

Role

UX Designer (Research, Visual Design, Interaction Design, User Testing)

Team

Team project

Duration

16 weeks
